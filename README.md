# Tic-Tac-Toe-Kivy
A simple Tic-Tac-Toe game in Python, with Kivy.

## Installation and runnig

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Kivy.

```bash
pip install Kivy
```

## Usage

Please make sure firstly you have a Python version under 3.8.
